# **Swipe List**

### What does it do
___

Fork and Rewrite edition.
___

Populates the dropdown list with the loaded swipes and adds buttons to switch to that swipe.


### Requirements
___

[LaLib Extension](https://github.com/LenAnderson/SillyTavern-LALib)
